git status：查看仓库状态
git log：打印仓库提交日志
git init：初始化Git仓库
git remote add origin git@github.com:wangbo-2022/python.git：关联远程仓库
git add 文件名：将文件添加到Git仓库
git commit -m "提交信息"：将添加的文件提交到仓库
git push origin master：将本地代码提交到github
git pull origin master：将github代码下载到本地

代码提交流程：
git add .
git commit -m "commit info"
git push origin master
